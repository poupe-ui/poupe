Examples Overview

This unified example demonstrates:

  • Surface utilities: Primary, secondary, tertiary, error, and container surfaces
  • Elevation system: Shadow z-levels (z1, z3) for depth
  • Scrim overlays: Semantic scrims (modal, drawer, base) with opacity modifiers
  • Z-index system: Semantic z-index utilities for layering
  • Fixed colors: Fixed surface combinations for consistent theming

How to build and view:

pnpm run build
# Then open examples/index.html in browser
